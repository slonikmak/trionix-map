# Архитектурная спецификация проекта (v4.0)

## Глобальная архитектура

Система разделена на два изолированных компонента:

1.  **[JFX-Map-Engine](./LIBRARY_SPEC.md)** (Библиотека) — Универсальный графический движок для отображения гео-данных. Не имеет зависимостей от бизнес-логики, баз данных или протоколов робота.

2.  **[ROV-Mission-Control](./APPLICATION_SPEC.md)** (Приложение) — Управляющая программа. Отвечает за хранение данных, связь с ТНПА, логику миссии и управление библиотекой.

## Быстрая навигация

### Библиотека (JFX-Map-Engine)

- Контракт `IGeoProjection` — система координат и проекция
- Контракт `ITileProvider` — поставщик тайлов
- Контракт `IMapLayer` — базовый слой карты
- Специализированные слои: `VectorLayer`, `GradientPathLayer`, `CachedBitmapLayer`
- Контракт `IMapInteractive` — обработка взаимодействий

### Приложение (ROV-Mission-Control)

- Подсистема данных (Project Manager, Data Providers)
- Подсистема телеметрии (Parser, Track Builder, State Monitor)
- Подсистема инженерных данных (DXF Importer)
